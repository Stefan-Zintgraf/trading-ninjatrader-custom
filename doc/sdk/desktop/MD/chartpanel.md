# ChartPanel

The **ChartPanel** class includes a range of properties related to the [panel](./chartpanels.md) on which the calling script resides. Each Panel has 3 independent [ChartScales](./chartscale.md): Left, Right, and Overlay.

![ChartPanel_1](https://cdn.sanity.io/images/1hlwceal/production/cf8a7cc24793305e26d007ab250fb26a66a55d80-639x525.png)

## [Methods and Properties](./chartpanel.md)

Parameter

Description

[ChartObjects](./chartobjects.md)

A collection of objects configured on the chart panel

[H](./h_height.md)

Indicates the height (in pixels) of the chart panel

[IsFocused](./chartpanel_isyaxisdisplayedleft.md)

Indicates the chart panel is currently in focus in the window

[IsYAxisDisplayedLeft](./chartpanel_isyaxisdisplayedleft.md)

Indicates the y\-axis is visible on the left side of the chart panel

[IsYAxisDisplayedOverlay](./isyaxisdisplayedoverlay_chartpanel.md)

Indicates any objects configured in the panel are using the Overlay scale justification

[IsYAxisDisplayedRight](./isyaxisdisplayedright_chartpanel.md)

Indicates the y\-axis is visible on the right side of the chart panel

[MaxValue](./maxvalue_chartpanel.md)

Indicates the maximum Y value of objects within the chart panel

[MinValue](./minvalue_chartpanel.md)

Indicates the minimum Y value of objects within the chart panel

[PanelIndex](./panelindex_chartpanel.md)

Indicates the index of the chart panel in the collection of configured panels

[Scales](./chartscale_chartpanel.md)

A collection of [ChartScale](./chartscale.md) objects corresponding to objects within the chart panel

[W](./w_width_chartpanel.md)

Indicates the width (in pixels) of the chart panel

[X](./x_coordinate_chartpanel.md)

Indicates the x\-coordinate on the chart canvas at which the chart panel begins

[Y](./y_coordinate_chartpanel.md)

Indicates the y\-coordinate on the chart canvas at which the chart panel begins
