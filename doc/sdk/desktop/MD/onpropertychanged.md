# OnPropertyChanged()

## [Definition](./onpropertychanged.md)

This method should be used any time you wish to repaint the column instead of calling **OnRender()** directly.

## [Method Return Value](./onpropertychanged.md)

This method does not return a value.

## [Syntax](./onpropertychanged.md)

`OnPropertyChanged()`

## [Parameters](./onpropertychanged.md)

This method does not require any parameters.

## [Examples](./onpropertychanged.md)

```csharp
// Repaint the SuperDOM column
OnPropertyChanged();

```
