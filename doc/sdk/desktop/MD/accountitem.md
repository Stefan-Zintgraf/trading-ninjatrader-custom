# AccountItem

# AccountItem

# Definition

Represents various account variables used to reflect values the status of the account. Each account connected in NinjaTrader will have it's own unique AccountItem values.

![note image](/_next/image?url=%2F_next%2Fstatic%2Fmedia%2FNote.57c6078c.svg&w=64&q=75)

## Note

For strategies, see also [OnAccountItemUpdate](./onaccountitemupdate.md). For other objects, you can also subscribe to the [AccountItemUpdate](./accountitemupdate.md) stream.
