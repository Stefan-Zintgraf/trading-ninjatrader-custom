# Using CancelOrder() method to cancel orders

When using NinjaTrader's Entry and Exit methods, the default behavior is to automatically expire them at the end of a bar unless they are resubmitted to keep them alive. Sometimes you may want more flexibility in this behavior and wish to submit orders as live\-until\-cancelled. When orders are submitted as live\-until\-cancelled, the way to cancel them is by using the [CancelOrder()](./cancelorder.md) method.

## [Key concepts in this example](./using_cancelorder_method_to_ca.md)

*   Submitting live\-until\-cancelled entry orders
*   Manually cancelling orders
